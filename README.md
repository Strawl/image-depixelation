# image-depixelation

Link to github page: https://github.com/Strawl/image-depixelation/

# how to use
1. run download.sh
```
chmod +x ./download.sh
./download.sh
```
2. install requirements
```
pip install -r requirements.txt
```
3 execute python script
```
python3 ./image-depixelation/main.py
```
